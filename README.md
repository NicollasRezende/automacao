# automacao
### basicamente utilizando uma biblioteca python que permite que voce automatize alguma coisa

# Exemplo
- fecha todas as janelas 
- cria um arquivo txt
- nomeia o arquivo
- escreve no arquivo
- close arquivo

### para usar este codigo e necessario instalar a biblioteca pyautogui

desta forma:

- pip install pyautogui
